# ข้าว ENT — แอปสั่งข้าว + เครดิต + กองกลาง

เว็บแอปสำหรับ resident ENT ปี 1: สั่งข้าวกลางวัน, ระบบเครดิตเติมเงิน (ติดลบได้),
เงินฝากอาจารย์, และบัญชีกองกลาง — แทน Google Sheet เดิม

- **มือถือเป็นหลัก**, ภาษาไทย
- **ไม่มีรหัสผ่านรายคน** — เปิดลิงก์แล้วแตะชื่อตัวเอง
- งานเงินทุกอย่าง (เติมเงิน/ปรับยอด/กองกลาง) ต้องใส่ **รหัสแอดมิน**
- ฟรี: Supabase + Vercel (ฟรีทั้งคู่ที่ขนาดรุ่น ~19 คน)

## ฟีเจอร์ (ตามแท็บล่าง)

| แท็บ | ทำอะไร |
| --- | --- |
| 🍚 สั่งข้าว | แตะชื่อ → ใส่ออเดอร์วันนี้ (OR/OPD, เมนู, ราคา) → เห็นบอร์ดรวมแบบเรียลไทม์ + เครดิตคงเหลือของตัวเอง |
| 💰 เครดิต | ยอดคงเหลือของทุกคน (เขียว=มีเครดิต / แดง=ติดหนี้) + ยอดใช้สัปดาห์นี้ |
| 📋 รวมออเดอร์ | รายการที่ต้องสั่งร้าน แยก OR/OPD + ปุ่มคัดลอกไปแชต + บันทึก “ใครสำรองจ่าย” (โรลเข้าเครดิต) |
| 🏦 กองกลาง | บัญชีเงินส่วนกลาง รายรับ/รายจ่าย + ยอดคงเหลือ |
| ⚙️ แอดมิน | เติมเงิน, ปรับยอด, จ่ายคืนสด, จัดการสมาชิก |

## ระบบเครดิตทำงานยังไง

`ยอดคงเหลือ = (เติมเงิน + เครดิตสำรองจ่าย + ปรับยอด) − ค่าข้าวที่สั่ง`

- ติดลบได้ = ติดหนี้ ค่อยทวงให้มาเติม
- **โรลเข้าเครดิต:** วันไหนใครสำรองจ่ายค่าข้าวที่ห้อง treatment ให้ไปที่ 📋 รวมออเดอร์ →
  “ใครสำรองจ่าย” → เลือกชื่อ → ระบบเพิ่มเครดิตให้เท่ายอดที่จ่าย (ไม่ต้องโอนสด)
  ค่าข้าวของคนนั้นยังถูกหักตามปกติ เงินเลยหักล้างกันพอดี
- **อาจารย์ไพบูลย์:** ใช้เงินฝากรายปี ค่าข้าวหักจากยอดฝาก ดูคงเหลือได้ที่แท็บเครดิต

## ตั้งค่าครั้งแรก (ทำครั้งเดียว ~10 นาที)

### 1) สร้างฐานข้อมูล Supabase (ฟรี)
1. สมัคร/เข้า https://supabase.com → **New project** (เลือก region สิงคโปร์ใกล้สุด)
2. ตั้งรหัสฐานข้อมูล (เก็บไว้) แล้วรอโปรเจกต์สร้างเสร็จ
3. เปิด **SQL Editor** → วางไฟล์ [`supabase/schema.sql`](supabase/schema.sql) ทั้งหมด → **Run**
4. เปิด SQL Editor อีกแท็บ → วาง [`supabase/seed.sql`](supabase/seed.sql) → **Run**
   (นำเข้ารายชื่อ, ออเดอร์สัปดาห์ 8–12 มิ.ย., ยอดกองกลาง, เงินฝากอาจารย์)
5. ไป **Project Settings → API** คัดลอก 2 ค่า:
   - `Project URL`
   - `anon public` key
   - `service_role` key (อันลับ อย่าเผยแพร่)

### 2) ใส่ค่าในเครื่อง (สำหรับรันทดสอบ)
คัดลอก `.env.local.example` เป็น `.env.local` แล้วเติมค่า:
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
ADMIN_PIN=ตั้งรหัสอะไรก็ได้
```
รันในเครื่อง: `npm install` แล้ว `npm run dev` → เปิด http://localhost:3000

### 3) ขึ้นออนไลน์ด้วย Vercel (ฟรี)
1. สมัคร/เข้า https://vercel.com → **Add New → Project**
2. อัปโหลดโค้ดนี้ (ผ่าน GitHub หรือ Vercel CLI)
3. ใส่ Environment Variables 4 ตัวเดียวกับข้างบน (`NEXT_PUBLIC_SUPABASE_URL`,
   `NEXT_PUBLIC_SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_ROLE_KEY`, `ADMIN_PIN`)
4. **Deploy** → ได้ลิงก์ เช่น `https://khao-ent.vercel.app` แชร์ให้รุ่นได้เลย
